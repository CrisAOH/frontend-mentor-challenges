# Frontend Mentor Challenges

This repository contains my solutions to various Frontend Mentor challenges.

The goal of this repository is to practice and improve my front-end development skills by recreating real-world UI designs.

## Repository Structure

Each challenge is located inside the challenges directory and organized by difficulty.

```bash
frontend-mentor
├── challenges
│   ├── newbie
│   ├── junior
│   ├── intermediate
│   ├── advanced
│   └── guru
└── README.md
```

Each challenge contains its own source code and README file.
